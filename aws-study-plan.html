AWS Cloud Practitioner 4-Week Study Plan

Created by Tufelo Minenhle Madlala to prepare for CLF-C02 certification

Week 1: Cloud Basics & Core Services

  • Learn cloud computing concepts & AWS global infrastructure
  • Explore EC2, S3, Lambda, IAM
  • Practice using AWS Skill Builder Labs
  • Take sample quiz from AWS

Week 2: Billing & Cost Management

  • Understand pricing models: On-demand, Reserved, Spot
  • Learn tools like Cost Explorer & Budgets
  • Study AWS Support Plans
  • Mock exam on Whizlabs or Tutorials Dojo

Week 3: Security & Monitoring Tools

  • Deepen knowledge of IAM, CloudTrail, CloudWatch
  • Explore databases & networking basics (VPC, RDS)
  • Review exam tips & test strategies
  • Take full-length practice test

Week 4: Final Prep

  • Revise weak areas and key services
  • Use flashcards for memorization
  • Take another full-length test
  • Prepare for exam day (calm + confident!)
